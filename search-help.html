Searching

Publication searching

The search box (search within this version) is on the left hand side .

Type a word or term into this box and click the Search button to search the entirety of the British Pharmacopoeia. The search will find articles and monographs that match all the terms entered. It is not case sensitive.

Advanced searching

The Advanced Search is accessible from a link below the Publication search and offers functions to narrow the scope of the search as well as support for boolean operators.

The Keyword and Title boxes allow searching of either the whole of the BP text or only monograph titles. The radio buttons below these boxes specify whether the search performs a boolean AND or a boolean OR when multiple terms are entered.

The 'Add Not Keyword' textbox can be used to specify words that if matched should be excluded from the search results. For instance searching for 'test' with a NOT Keyword of 'tube' will show all results matching 'test', that don't match 'tube'.

A dropdown list on the Advanced Search form also allows searches to be restricted to British, European, or Internationally harmonised monographs, and to specific sections of BP.

For example, to search a specific section, enter your terms in the search fields and select the required section from the dropdown list.

It is possible to select multiple items from the 'Filter Sections' dropdown to broaden the search.
